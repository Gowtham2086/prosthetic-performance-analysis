# Prosthetic Performance Analysis

This project analyzes the relationship between prosthetic accuracy and user comfort using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Streamlit

## Key Insight
Accuracy has only a weak impact (~10%) on user comfort.

python analysis.py
